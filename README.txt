==================================================| PROJETO CADASTRO DE CANDIDATO TEX|========================================================


- Descrição do projeto

Projeto realizado para seleção, com o obetivo de realizar o cadastro de um candidato onde foi informado o passo a passo para realização do cadastro.
O intuito e testar as funcionalidades da pagina relacionado ao cadastro do candidato, o projeto foi implementado na tecnica de desenvolvimento agil BDD,
foi utilizado o framework cypress pois é o Framework que venho me aperfeiçoando e estudando junto com a linguagem Javascript a muito tempo.

- Funcionalidades do projeto

O teste tem a funcionalidade de realizar o cadastro de candidatos, de forma que seja possivel realizar o cadastro de candidatos diferentes preenchendo
todos os  campos.

- Tecnologias utilizadas

Nesse projeto foram utilizadas as seguintes tecnologias:

- Visual Studio Code;
- Linguagem JavaScript;
- Nodejs;
- Framework Cypress;
- Cucumber;
- Utilizado a biblioteca Faker;
- Projeto no padrão Page Object;

- Implementações futuras

Poderia ser implementado futuramente:

- mensagem informando que o cadastro foi realizado com sucesso;
- campo para informar outras tecnologias;
- incluir * nos campos que devem ser preenchidos;

testes que podem ser realizados:

- cadastro sem nome do candidato: o cadastro não seria realizado;
- cadastro sem sobrenome do candidato: o cadastro não seria realizado;
- cadastro sem informação de email: o cadastro não seria realizado;
- cadastro sem experiências: o cadastro não seria realizado;
- cadastro sem senioridade: o cadastro será realizado;
- cadastro sem tecnologias: o cadastro será realizado;
